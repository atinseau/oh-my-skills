# Commands

Les commandes shell distribuées par `oh-my-skills` vivent dans ce dossier.

- Chaque fichier `*.sh` est sourcé automatiquement dans le shell utilisateur
- Les sous-dossiers sont autorisés (ex: `oms-cli/oms.sh`) et sont sourcés récursivement
- Les commandes sont copiées dans `~/.oh-my-skills/commands/` pendant l’installation
